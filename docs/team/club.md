# 学生社团

???+ tip "动态更新"

    本页面为动态更新界面

加入方式：一般可以于每学年社团纳新环节加入社团。但附中对社团管理并非特别严格，部分社团在非纳新时间也会接受新同学的加入，具体情况可以咨询社长。

社团换届：社长任期为一学年，每学年结束时换届。如无特殊情况，一般由高二学生担任社长职务，但也可由高一或高三学生担任。

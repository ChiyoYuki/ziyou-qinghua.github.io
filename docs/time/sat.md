# 校本课与社团课

## 校本课

- 时间：
  - 高一：每周六 14:00-15:30
  - 高二：每周四 第 8-9 节课
- 地点：校内各教室
- 毕业要求：高一至少参加一次，高二必须参加两次
- 参加条件：高一和高二，选修相应校本课程
- 选课时间：每学期第三周前，请关注学委通知
